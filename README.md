# Dynamic Information Saver

This project helps you save and organize information in a simple web page.
You can add table data, create a visual report, and download the result as a PDF.

## What this system does

1. Lets you enter information in a table.
2. Saves your data in the browser.
3. Creates simple charts from your data.
4. Lets you download the report as a PDF.
5. Lets you export table data when needed.

## How to use it

### Step 1: Install the app

Open a terminal in this folder and run:

```powershell
npm install
```

### Step 2: Start the app

Run:

```powershell
npm start
```

### Step 3: Open it in the browser

After the app starts, open:

```text
http://localhost:3000
```

### Step 4: Create your table

- Click **Generate Table**.
- Add or change the information you want to store.
- The data stays saved while you keep using the app.

### Step 5: Make a report

- Click **Create Graph**.
- The app shows a simple report using charts.
- The report is made from the same data in your table.

### Step 6: Save the report

- Click **Save Graph PDF** to download the report as a PDF.
- Click **Download PDF** if you want the table version as a PDF.
- Click **Save to Excel** if you want the table data in Excel format.

## Files in this project

- `index.html` — the page you see in the browser
- `styles.css` — the look and layout
- `script.js` — the main app logic
- `server.js` — starts the local web server

## Simple idea of the project

Think of this app like a small digital notebook:

- You enter data.
- The app stores it.
- The app turns it into charts.
- You save the final report as a PDF.

## If something does not open

- Make sure Node.js is installed.
- Run `npm install` again.
- Run `npm start` again.
- Refresh the browser page.

## Short version

This app is for saving data, seeing it as a simple report, and downloading it in PDF form.
