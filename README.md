# Dynamic-Information-Saver

Saves information dynamically and efficiently.

## Run With Node.js

1. Install Node.js LTS if it is not already installed.
2. Open a terminal in this folder.
3. Install dependencies:
   ```powershell
   npm install
   ```
4. Start the server:
   ```powershell
   npm start
   ```
5. Open http://localhost:3000 in your browser.

## Current Structure

- `index.html` for markup
- `styles.css` for layout and styling
- `script.js` for table behavior, persistence, search, and PDF export
- `server.js` for the Node.js backend
